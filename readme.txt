== Description ==

Add QQ Yun IME to your WordPress.
为你的 WordPress 添加 QQ 云输入法。

= 支持英语、拼音、五笔输入 =
= 支持繁简体输入 =


QQ云输入法现在可以应用于所有主流浏览器上，如 IE 6-8，Firefox 1-4，Opera 9-10，Safari 3-5，Chrome 1-6，包括一些为智能手机设计的高级浏览器。

QQ云输入法会根据所有用户的输入习惯来调整候选词的排列位置，实现候选词的智能排列，提高用户的输入体验。

== Installation ==

1. 上传本插件到 /wp-content/plugins/ 目录；
2. 在 “插件” 页面激活 QQ Yun IME；
3. 在 “QQ 云输入法设置” 页面配置本插件。

本插件可以使用 `<?php qq_yun_ime(); ?>` 进行调用
